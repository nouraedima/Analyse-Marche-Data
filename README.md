#  Analyse du Marché Data 2026

Projet personnelle visant à analyser les opportunités de postes en Data (Analyst, Scientist, Engineer) en France.

#  Technologies utilisées
**Python** (Cœur du projet)
* **Selenium** (Web Scraping et Automatisation)
* **Pandas** (Traitement et nettoyage de données)
* **Matplotlib** (Visualisation de données)

# Résultats
Le script génère un dashboard comparatif montrant la répartition entre les CDI, les stages et les alternances pour aider à cibler la recherche d'emploi.
